This is Jatin Tahiliani's code for the final project for CS598: DL4H which is the reproducibility research of the original paper: Uncovering Knowledge Gaps in Radiology Report Generation Models through Knowledge Graphs by Xiaoman Zhang, Julian N. Acosta, Hong-Yu Zhou, Pranav Rajpurkar

The results were generated via colab. The dataset used was a custom, synthetic csv file of 100 various radiology reports.
